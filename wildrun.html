wild run

Wild Run is a free game app for mobile devices. In the game, the player must survive as long as possible in the jungle by tapping left and right on the screen while dodging some obstacles (fallen rocks, gusting winds and stinging bees). The secondary mechanics are the power ups such as magnets, health potions and invincible shields to aid the player and increase their score. The goal of this game is to achieve the highest score on the leaderboards and to unlock all the characters.

I created most graphic components, including character design and interface design, as well as all the promotional materials.

process

Wild Run was already in development when I joined the team, therefore I contributed to this game for 3 months. I was mainly responsible in creating all the character sprites. I created all the sprites in Adobe Illustrator. In addition I helped refine and add on to the user interface of the game to make the game look more appealing. I have also refined the trailer using Adobe Premiere and Adobe After Effects. Other tasks I were to do was to create all the arts requested by our publisher, Thumbspire. Therefore I had to create the screenshots and cross promotion assets for both iOS and Andriod sizes. I also designed the social assets to put on Thumbspire's Facebook and Twitter page.
